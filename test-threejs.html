🐛 Troubleshooting

Particles not showing?

1. Check if WebGL is enabled: Visit chrome://gpu

2. Open browser console (F12) and check for errors

3. Try disabling and re-enabling hardware acceleration

Low FPS on mobile?

1. Reduce particle count further (edit ParticleNetwork.tsx)

2. Change: const particleCount = isMobile ? 40 : 150;

3. Test on actual device (not just Chrome DevTools mobile mode)

Tilt cards not working?

1. Verify package installed: npm list react-parallax-tilt

2. Clear cache and rebuild: rm -rf node_modules && npm install

3. Check for conflicting CSS that might block pointer events
